# nexu をシェアしてポイントを獲得する方法

`nexu` には追加ポイントを獲得できるリワード制度があります。毎日のチェックイン、オープンソースプロジェクトの応援、そして SNS で `nexu` をシェアすることで、追加ポイントを受け取れます。獲得したポイントは、より多くの機能や体験を続けて使うために利用できます。

## 獲得できるポイント

現在受け取れるポイントは次のとおりです。

- 毎日チェックイン: +100 ポイント、1 日 1 回まで
- GitHub Star: +300 ポイント、1 回のみ
- X にシェア: +200 ポイント、週 1 回まで
- Reddit に投稿: +200 ポイント、週 1 回まで
- Xiaohongshu に投稿: +200 ポイント、週 1 回まで
- LinkedIn に投稿: +200 ポイント、週 1 回まで
- Jike に投稿: +200 ポイント、週 1 回まで
- WeChat にシェア: +100 ポイント、週 1 回まで
- Feishu にシェア: +100 ポイント、週 1 回まで
- Facebook にシェア: +200 ポイント、週 1 回まで
- WhatsApp にシェア: +200 ポイント、週 1 回まで

## ポイントの受け取り方

`nexu` クライアント内の `Rewards` ページで、すべてのリワードタスクを確認できます。タスクを完了したあと、ページに戻って確認すると、対応するポイントを受け取れます。

タスクの種類ごとの受け取り方は次のとおりです。

## 1. 毎日チェックイン

`Rewards` ページに入り、その日のチェックイン確認を完了すると 100 ポイントを受け取れます。

毎日 `nexu` を使っているなら、このタスクがいちばん安定したポイント源です。継続して使うほど、追加ポイントも安定して獲得できます。

## 2. GitHub で Star を付ける

`nexu` を気に入っていただけたら、ぜひ GitHub で Star を付けてください。完了後にクライアントへ戻って確認すると、300 ポイントを受け取れます。

これは 1 回限りのリワードです。`nexu` にとって GitHub Star は単なる応援ではなく、プロジェクトの認知拡大、貢献者の参加、継続的な改善につながる大切なシグナルでもあります。

## 3. SNS で nexu をシェアする

`nexu` を SNS でシェアしてポイントを獲得することもできます。現在対応しているプラットフォームは次のとおりです。

- X
- Reddit
- Xiaohongshu
- LinkedIn
- Jike
- WeChat
- Feishu
- Facebook
- WhatsApp

プラットフォームによって、タスクの流れは少し異なります。

- リンク系プラットフォーム: タスクをクリックすると対象ページが開き、シェア後に `nexu` に戻って確認します
- 投稿系プラットフォーム: たとえば X では、あらかじめ内容が入った投稿画面が開き、投稿後に戻って確認します
- 画像系プラットフォーム: たとえば Xiaohongshu、Jike、WeChat、Feishu では、`nexu` が先にシェアカードと投稿文を用意し、投稿後に戻って確認します

## 注意事項

ポイントを受け取るときは、次のルールに注意してください。

- 毎日チェックインは自然日ベースで、1 日 1 回までです
- 毎週のシェアタスクは週 1 回までです
- GitHub Star は 1 回限りのリワードです
- タスク完了後は `Rewards` ページに戻って手動で確認する必要があります
- リワード項目、ポイント数、受け取り頻度が今後変更される場合は、`nexu` クライアント内の `Rewards` ページ表示が基準になります

## よくある質問

**Q: シェアを完了したのに、まだポイントが付与されません。**

タスク完了後に `Rewards` ページへ戻って確認する必要があります。確認が完了して初めてポイントが付与されます。

**Q: 一部のプラットフォームで先に画像がダウンロードされるのはなぜですか？**

Xiaohongshu、Jike、WeChat、Feishu などは画像付きのシェアに向いています。`nexu` があらかじめ画像と投稿文を用意してくれるので、手動で素材を整理する手間を減らせます。

**Q: リワードのルールは変更されますか？**

変更される可能性はあります。各チャネルの実際の効果やキャンペーン内容に応じて更新されることがあります。変更がある場合は、クライアント内の表示を基準にしてください。
