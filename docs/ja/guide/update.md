# アップデートガイド

## 方法1：アプリ内から直接アップデート（推奨）

1. Nexu を開く
2. トップメニューバーの **Nexu** をクリック
3. **Check for Updates** を選択
4. 新しいバージョンがある場合は、案内に従ってアップデートを完了する

> 最もシンプルな方法です。まずこちらをお試しください。

![Check for Updates](/assets/nexu-check-for-updates.webp)

---
