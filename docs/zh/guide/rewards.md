# 如何通过分享 nexu 获得积分

`nexu` 提供了一套额外积分奖励机制。你可以通过每日打卡、支持开源项目，以及在社交平台分享 `nexu`，获得额外积分，用于继续体验更多能力。

## 你可以获得哪些积分

目前可领取的积分如下：

- 每日打卡：+100 积分，每天可领取一次
- GitHub Star：+300 积分，仅限一次
- 分享到 X：+200 积分，每周可领取一次
- 发帖到 Reddit：+200 积分，每周可领取一次
- 发帖到小红书：+200 积分，每周可领取一次
- 发帖到领英（LinkedIn）：+200 积分，每周可领取一次
- 发帖到即刻：+200 积分，每周可领取一次
- 分享到微信：+100 积分，每周可领取一次
- 分享到飞书：+100 积分，每周可领取一次
- 分享到 Facebook：+200 积分，每周可领取一次
- 分享到 WhatsApp：+200 积分，每周可领取一次

## 如何领取积分

你可以在 `nexu` 客户端的 `Rewards` 页面查看全部奖励任务。完成任务后，回到页面确认，即可领取对应积分。

不同任务的领取方式如下。

## 1. 每日打卡

进入 `Rewards` 页面，完成当日打卡确认后，即可领取 100 积分。

如果你每天都在使用 `nexu`，这个任务是最稳定的积分来源。只要保持活跃，就能持续获得额外积分。

## 2. 在 GitHub 上为我们点 Star

如果你认可 `nexu`，欢迎前往 GitHub 为我们点一个 Star。完成后回到客户端确认，即可获得 300 积分。

这是一次性奖励。对 `nexu` 来说，GitHub Star 不只是支持，也是开源项目获得关注、吸引贡献者、持续迭代的重要信号。

## 3. 在社交平台分享 nexu

你也可以通过在社交平台分享 `nexu` 来获得积分奖励。当前支持的平台包括：

- X
- Reddit
- 小红书
- 领英（LinkedIn）
- 即刻
- 微信
- 飞书
- Facebook
- WhatsApp

不同平台的任务流程会有一点区别：

- 链接类平台：点击任务后会打开对应页面，完成分享后返回 `nexu` 确认即可
- 发帖类平台：例如 X，会打开预填内容的发帖页，发布后返回确认即可
- 图片类平台：例如小红书、即刻、微信、飞书，`nexu` 会先准备分享卡和配文，你发布后再返回确认领取积分

## 注意事项

领取积分时，请留意以下规则：

- 每日打卡按自然日计算，每天可领取一次
- 每周分享任务每周可领取一次
- GitHub Star 为一次性奖励
- 完成任务后，需要返回 `Rewards` 页面手动确认
- 如果后续奖励项、积分值或频次发生调整，请以 `nexu` 客户端内 `Rewards` 页面显示为准

## 常见问题

**Q: 分享完成后，为什么还没有拿到积分？**

完成任务后，需要回到 `Rewards` 页面点击确认，积分才会发放。

**Q: 为什么有些平台会先下载图片？**

小红书、即刻、微信、飞书等平台更适合使用分享卡进行发布。`nexu` 会提前帮你准备图片和配文，减少你手动整理内容的步骤。

**Q: 奖励规则会调整吗？**

有可能。我们会根据不同渠道的实际效果和活动安排进行更新。如果有变化，请以客户端内显示为准。
